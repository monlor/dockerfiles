## 项目

radiance-bot，oracle抢机

## 安装

#### 环境变量

获取用户信息：https://t.me/radiance_helper_bot

* TG_BOT_USERNME
* TG_BOT_PASSWORD

### 映射配置文件

* `/app/oracle_config`: oracle配置
* `/app/azure_config`: azure配置

### 映射key

* `/app/oracle.key`
* `/app/azure.key`
